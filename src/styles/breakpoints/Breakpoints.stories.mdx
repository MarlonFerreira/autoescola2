# Breakpoints

Breakpoints for responsive layout:

<table>
  <tr>
    <th>Extra Small (xs)</th>
    <td>Default</td>
  </tr>
  <tr>
    <th>Small (sm)</th>
    <td>>= {BreakpointsSizes.sm}px</td>
  </tr>
  <tr>
    <th>Medium (md)</th>
    <td>>= {BreakpointsSizes.md}px</td>
  </tr>
  <tr>
    <th>Large (lg)</th>
    <td>>= {BreakpointsSizes.lg}px</td>
  </tr>
  <tr>
    <th>Extra Large (xl)</th>
    <td>>= {BreakpointsSizes.xl}px</td>
  </tr>
  <tr>
    <th>Extra Extra Large (sm)</th>
    <td>>= {BreakpointsSizes.xxl}px</td>
    <td>Sem uso</td>
  </tr>
</table>

Using in styled component:

```js
import { breakAt, BreakpointSizes } from Breakpoints;

const Title = styled.h1`
    font-size: 1rem;

    ${breakAt(BreakpointSizes.md)} {
        font-size: 2rem;
    }
`
```
